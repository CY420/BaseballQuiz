# ⚾ Baseball & CPBL Quiz (棒球與中職知識大挑戰)

這是一個使用 **SwiftUI** 開發的 iOS 選擇題 App。結合了標準棒球規則、歷史紀錄以及中華職棒（CPBL）的相關知識，讓球迷或想要了解棒球的使用者能透過遊戲測試自己的實力。

此專案同時也是 SwiftUI 的練習作品，完整實作了狀態管理、資料綁定以及客製化的 UI/UX 設計。

## ✨ 功能特色 (Features)

### 📌 基本功能 (Basic Requirements)
- **選擇題測驗**：每題提供四個選項，涵蓋基礎規則與 CPBL 歷史。
- **自訂資料型別**：使用 Struct `Question` 嚴謹定義題庫結構（題目、正確答案、錯誤答案）。
- **隨機抽題機制**：總題庫大於 10 題，每次遊戲隨機抽出 10 題，確保每次遊玩的題目與順序皆不相同（使用 `Array.shuffle()`）。
- **即時狀態顯示**：同一頁面動態顯示「目前題號（如：3/10）」與「當前得分」。
- **完整遊戲迴圈**：答完 10 題後進入結算畫面，並提供「再玩一次」功能，點擊後重置所有狀態與題庫。

### 🔥 進階與特殊功能 (Advanced Features)
- **選項順序隨機**：不只題目隨機，每題的 4 個選項位置也會在初始化時隨機打亂，防止背答案。
- **特殊 Combo 計分機制**：
  - 基礎計分：答對 +10 分。
  - **連擊獎勵 (Combo)**：連續答對 3 題以上，每答對一題大幅加分 (+30 分)，並顯示 🔥 視覺提示。
  - **答錯懲罰**：答錯時扣 10 分，中斷連擊紀錄（分數具備防護機制，最低為 0 分不扣至負數）。
- **客製化 UI 設計**：
  - 導入高質感的自訂全螢幕背景圖片 (`.scaledToFill()` 與 `.ignoresSafeArea()`)。
  - 運用毛玻璃效果 (`.opacity` 與遮罩) 確保題目與選項在複雜背景下依然清晰易讀。
  - 針對卡片外觀設計帶有顏色的外框與陰影，提升點擊回饋感與視覺層次。
